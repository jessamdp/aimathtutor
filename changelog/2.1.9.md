# AIMathTutor Changelog — Version 2.1.9

## Changed

- JDK version updated from 21 to 25
- Quarkus version updated from 3.27.2 to 3.33.1.
  - Switched to `quarkus` packaging
  - Updated `quarkus-junit` dependencies
  - Updated plugin configurations
- Vaadin version updated from 25.0.6 to 25.1.0.

## Removed

- Removed ubi-based Dockerfile
  - eclipse-temurin does not offer ubi9-based JRE 25 images
  - ubi10-based images have compatibility issues with certain CPUs

## Security

- Removed pinned version overrides for Jackson and Vert.x dependencies
- Upgraded Netty to 4.1.132.Final to address HTTP request smuggling vulnerabilities
