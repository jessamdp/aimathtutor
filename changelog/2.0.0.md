
# AIMathTutor Changelog — Version 2.0.0

## Added

- New admin configuration view for runtime AI provider/model/settings management.

## Changed

- JDK/JRE versions updated from 21 to 25 (previous LTS to latest LTS).
- Vaadin version updated from 24.9.3 to 24.9.5.
- Unpublished exercises excluded when computing exercise count for a given lesson in the admin lessons view.
- Minor improvements to individual comment layout in exercise view.

## Removed

- Config Properties for settings that can be managed through the new AI admin configuration view.

## Fixed

- Comments which are set to hidden through the admin panel are no longer displayed in the comments panel.
- Lessons which only contain unpublished exercises are no longer displayed to users on the lessons view.
