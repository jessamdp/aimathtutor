# AIMathTutor Changelog — Version 2.1.7

## Security

- Upgraded minimatch to fix ReDoS vulnerability via repeated wildcards with non-matching literals in patterns.
- Upgraded Rollup 4 to fix arbitrary file write via path traversal vulnerability.
- Upgraded serialize-javascript to fix RCE vulnerability via RegExp.flags and Date.prototype.toISOString.
- Upgraded Jackson fasterxml 2.x to 2.21.1 to fix async parser DoS vulnerability.
- Upgraded Jackson tools 3.x to 3.1.0 to fix async parser DoS vulnerability.
