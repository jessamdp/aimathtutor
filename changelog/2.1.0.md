
# AIMathTutor Changelog — Version 2.1.0

## Added

- Retry mechanism for AI tutor responses, automatically retrying failed requests for improved reliability.
- Connection and read timeout settings for Ollama integration, configurable via environment variables.

## Changed

- Default Ollama model changed to llama3.2:3b.
- Default AI response token limit increased from 1000 to 2000 across all providers to prevent incomplete responses.
- Improved AI chat panel responsiveness with better typing indicator management during async operations.

## Fixed

- AI tutor now gracefully handles incomplete or malformed AI provider responses, displaying clean feedback instead of raw technical data.
- Typing indicator in AI chat panel no longer behaves unexpectedly during rapid interactions.
